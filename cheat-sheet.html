Enumerable Cheat Sheet

.each

Each iterates through a data structure to access each element. Imagine an array of strings. Each is mostly used for accessing elements. You can modify and return a single element from the data structure without modifying the data structure itself.

.map

Map iterates through a data structure and effectuate the block. Lets say you have an array of numbers and you want to do something to each element; you then use .map to access and change the elements.

.slice

Slice is a method used to delete elements from an array. Slice does not only delete an element. It also prints the element you chose to delete. You can also choose a range of elements to delete. At the end all the elements deleted are printed as a result.

.group_by

The “group_by” method is, essentially, a way to collect all of the things that are similar in an array and group them together in a hash.
