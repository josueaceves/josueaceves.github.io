.map method

The map method runs a block for each element in an array. It is used in enumerator objects. We commonly effectuate calculations or changes inside arrays. For example, lets say that you want to add a number to every element inside an array, you would then use the map method. The map method iterates through an array, element by element until it goes through each elemet and finishes. At the end, map returns the outcome based on the instructions addressed inside the block of map. Here is a quick example:

array = [10 ,20 , 30, 40, 50, 60] --> here we have an array containing numbers. lets say those numbers represented the age of artfacts in a museum two years ago, so now its time to add 2 years to each artfact for the array to represent the current ago. To do this we need to use the map method. array.map{|x| x + 2}.

After the map method finishes the iteration, it returns the outcome in an array. This can be very helpful to alot of us, but imagine if you actually wanted to mutate the original array definitely.
